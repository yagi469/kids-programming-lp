## 3. 実装計画 (Technical Implementation)

### 技術スタック
- **HTML5**: セマンティックなマークアップ。
- **CSS Framework**: Tailwind CSS (CDN版を使用)。
- **Font**: Google Fonts (`Zen Maru Gothic` for headings/friendly feel, `Inter` for body).
- **Icons**: FontAwesome or Heroicons (SVG).

### デザインテーマ
- **メインカラー**: オレンジ (元気、楽しさ、創造性)
    - Primary: `#F97316` (Orange-500)
    - Secondary: `#FFEDD5` (Orange-100)
    - Accent: `#3B82F6` (Blue-500, for trust/contrast)
- **雰囲気**: ポップで親しみやすいが、安っぽくならない「プレミアムな楽しさ」。
- **あしらい**: 角丸、ドロップシャドウ、ふわっとしたアニメーション。

### ファイル構成
- `index.html`: 全てのコードを含む（Tailwind Script設定含む）。
- `assets/`: 画像ファイル格納用。

### 画像生成予定 (Generate Image)
1.  **Hero Image**: 子供がPCで目を輝かせてプログラミングしている様子。親御さんが優しく見守っている。
2.  **Problem Image**: ゲームばかりしている子供の後ろ姿（少し暗め）。
3.  **Solution Image**: スクラッチのブロックを組み立てているイメージ（抽象的または画面風）。
4.  **Classroom/Trust Image**: 明るい教室で講師と生徒が話している様子。
